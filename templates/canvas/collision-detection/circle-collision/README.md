# Circle Collision Detection

Simple collision detection between two circles

## Usage

1.  Install dependencies:

        yarn

    or

        npm install

2.  Run webpack:

        npm start

Credits to christopher4lis for the [canvas boilerplate](https://github.com/christopher4lis/canvas-boilerplate)
